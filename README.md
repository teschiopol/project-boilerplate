# Vue 3 + TypeScript + Vite + PrimeVue 4

## Requirements

- node v20+

```bash
npm i
```

## Usage

For development

```bash
npm run dev
```

For production
```bash
npm run build
npm run preview
```

## Resources

- [Vue 3](https://vuejs.org/)

- [Typescript](https://www.typescriptlang.org/)

- [Vite](https://vitejs.dev/)
